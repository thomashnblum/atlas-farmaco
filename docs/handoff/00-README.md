# Handoff: Atlas Fármaco

O Atlas Fármaco é uma aplicação web interativa focada em psicofarmacologia e psiquiatria clínica. Ele serve como um catálogo visual e técnico de moléculas psicotrópicas (como antidepressivos, estabilizadores de humor e antipsicóticos), mapeando seus perfis farmacodinâmicos (afinidade por receptores), farmacocinéticos (interações com enzimas CYP/UGT), indicações clínicas, dosagens e efeitos colaterais. É projetado para auxiliar profissionais de saúde mental e estudantes na consulta rápida e detalhada de dados essenciais para prescrição e manejo de interações medicamentosas.
