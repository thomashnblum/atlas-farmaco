# Atlas Farmaco

> Catálogo interativo de psicofármacos para consulta clínica e educacional.

## Sobre o projeto

O Atlas Farmaco é um aplicativo web de consulta a medicamentos utilizados na psiquiatria, voltado para profissionais de saúde, estudantes e curiosos. Reúne informações sobre mecanismo de ação, indicações clínicas, usos off-label, efeitos adversos e interações dos principais psicofármacos disponíveis no Brasil.

## Funcionalidades

- Catálogo organizado por categoria (antidepressivos, antipsicóticos, estabilizadores de humor, ansiolíticos, estimulantes...)
- Comparação lado a lado entre medicamentos
- Indicações tradicionais e usos off-label documentados
- Visualização de interações farmacológicas

## Tecnologias

React · TypeScript · Google AI Studio (Gemini)

## Como rodar localmente

**Pré-requisito:** Node.js

1. `npm install`
2. Crie um arquivo `.env.local` na raiz com: `GEMINI_API_KEY=sua_chave_aqui`
3. `npm run dev`

## Aviso

Este aplicativo tem caráter **informativo e educacional**. Não substitui orientação, diagnóstico ou prescrição médica profissional.
